# Soundboard

A client-side soundboard web app with a QWERTY keyboard UI (desktop) and a responsive grid on mobile. Built with Vite, Vue 3, TypeScript, Tailwind CSS, and GSAP. Keys without a `soundUrl` in config are disabled.

## Features

- **QWERTY keyboard** (A–Z + Space) on desktop; each key shows a letter and sound label
- **Physical key support**: press the matching key to play a sound
- **Preloaded WAVs**: sounds load up front; loading state with spinner until ready
- **Overlapping playback**: multiple sounds (and repeated same key) can play at once
- **Progress overlays**: semi-transparent bar on each key fills by playback progress (GSAP)
- **Responsive**: mobile shows a grid of sound buttons; works on iPhone

## Setup

```bash
pnpm install
```

Or open the repo in a **Dev Container** (VS Code / Cursor): the container has Node 22 and pnpm; `pnpm install` runs automatically after the container is created. Port 5173 is forwarded for the dev server.

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview   # preview production build
```

## Sounds

Place your WAV files (≤30s) in `public/sounds/`:

- `a.wav` … `z.wav` for letter keys
- `space.wav` for the space bar

Placeholder (silent) WAVs are generated by:

```bash
node scripts/generate-placeholder-wav.js
```

Edit `src/lib/keys.ts` to change labels, add keys, or set `soundUrl` on a key to enable it (omit `soundUrl` to leave a key disabled). Run `pnpm run typecheck` to type-check.
